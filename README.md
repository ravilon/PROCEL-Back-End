# PROCEL-Back-End
Repositório destinado aos artefatos relacionados a estrutura do banco e arquitetura geral do procel

## Setup do dotenv - DataHarvester

### DB
PG_HOST=[hostname]
PG_PORT=[port]
PG_DATABASE=[database]
PG_USER=[dbuser]
PG_PASSWORD=[dbpassword]

### API
COBALTO_URL=https://cobalto.ufpel.edu.br/servicosgerais/consultas/salasDeAula/listaSalas/
COBALTO_TIMEOUT=60
